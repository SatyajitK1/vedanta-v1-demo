# 🎯 KPMG Visual Identity — Designer Brief

## 1. Logo Usage (CRITICAL)

* The logo is the **primary brand asset** → must always be treated carefully.
* Two main variants:

  * **White logo** → use on **dark backgrounds/images**
  * **Blue logo** → use on **light backgrounds / white**
* Never distort, recolor, or modify the logo.

---

## 2. Color System

### 🔵 Primary Colors (Core Brand)

* **KPMG Blue**

  * HEX: `#00338D`
  * RGB: 0 / 51 / 141
* **Medium Blue**

  * HEX: `#005EB8`
* **Light Blue**

  * HEX: `#0091DA`

👉 These should dominate the design.

---

### 🟣 Secondary Colors (Support Palette)

* **Violet** → `#483698`
* **Purple** → `#470A68`
* **Light Purple** → `#6D2077`
* **Green** → `#00A3A1`

👉 Use sparingly for accents, charts, highlights.

---

### 🎨 Color Composition Rule

* When combining colors:

  * Use **2 colors max**
  * Maintain a **40% / 60% split**
  * Always keep **white text + white logo** on colored backgrounds 

---

## 3. Typography System

### Primary Font Hierarchy

1. **KPMG Font**

   * Used for **Headlines**
   * Weights:

     * Extralight
     * Thin
     * Light

2. **Univers**

   * Used for **Subheadings + Body**
   * Weights:

     * 45 Light
     * 55 Roman
     * 65 Bold
     * 75 Black

3. **Arial**

   * Used for:

     * Web
     * PowerPoint
     * Emails
   * Regular + Bold

👉 Rule:

* Headlines = KPMG Font
* Body = Univers / Arial

---

## 4. Layout System (Very Important for UI/Website)

There are **3 layout groups (9 total variations)**:

### Group 1 — Vertical Image Layout

* Use when image is vertical
* Pattern:

  * **Image on right**
  * **Text on left (white text on dark image)**

---

### Group 2 — Single Image Layout

* Light, minimal layout
* Pattern:

  * **KPMG Blue bar on left**
  * Light background image
  * Blue text + logo

---

### Group 3 — No Image Layout

* Pure graphic layout
* Pattern:

  * Use **color blocks**
  * Combine 2 brand colors (40/60 rule)
  * White text + logo

👉 No layout is preferred — choose based on content type 

---

## 5. Imagery Guidelines

Use **consistent visual styles**:

* **Photography** → real-world, professional, corporate
* **Illustrations** → clean, minimal, modern
* **Textures** → abstract, subtle brand-aligned
* **Content Photos** → people, business environments

👉 Goal: consistent, premium, professional feel

---

## 6. Charts & Infographics

* Must:

  * Be **clean and minimal**
  * Use brand colors
  * Simplify complex data
* Two formats:

  * **Charts** → for data clarity
  * **Infographics** → for storytelling

👉 Example insight from PDF:

* Use visuals to explain **complex ideas simply** (not cluttered) 

---

## 7. Design Patterns to Follow

### Pattern 1

* Dark image background
* White text
* Vertical element on right

### Pattern 2

* Light image
* Blue accent bar (left)
* Blue text/logo

### Pattern 3

* Two-tone color block
* White text + logo

---

## 8. Overall Design Principles

* Clean, corporate, premium
* Minimal clutter
* Strong use of blue
* Consistent spacing and alignment
* Data-heavy visuals should feel **simple**

---


* Follow **KPMG brand colors strictly**
* Use **KPMG Blue as primary**
* Maintain **strict typography hierarchy**
* Stick to **3 layout systems**
* Use **white logo on dark / blue logo on light**
* Keep designs **minimal + structured**
* Charts = **simple, clean, brand-colored**
* Follow **40/60 color split rule**
